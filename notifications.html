Notifications

Telling the user something happened — without yanking them out of their flow. Trigger each one. Plain CSS/JS on the left, GSAP on the right.

Plain CSS / vanilla JS GSAP

1 · Toast / snackbar

A transient message slides in, then dismisses itself.

Plain

A .show class triggers a CSS slide + fade.

GSAP

A GSAP timeline slides in, waits, and slides out — all in one chain.

Plain — CSS + JS

.toast {
  transform: translateY(90px); opacity: 0;
  transition: transform .4s, opacity .4s;
}
.toast.show { transform: translateY(0); opacity: 1; }
// JS: add .show, then remove after 2.5s

GSAP — timeline

gsap.timeline()
  .from(toast, { y: 90, opacity: 0,
        ease: 'back.out(1.7)' })
  .to(toast, { y: 90, opacity: 0,
        delay: 2.2, onComplete: remove });

2 · Badge count update

The notification count ticks up with a little pop.

Plain · CSS pop
1

A CSS keyframe scales the badge when the number changes.

GSAP
1

GSAP pops the badge and swings the bell — a coordinated reaction.

Plain — JS

badge.textContent = ++count;
badge.classList.remove('pop');
void badge.offsetWidth;   // restart anim
badge.classList.add('pop');

GSAP — JS

badge.textContent = ++count;
gsap.fromTo(badge, { scale: 1.6 },
  { scale: 1, ease: 'back.out(4)' });
gsap.fromTo(bell, { rotation: -14 },
  { rotation: 0, ease: 'elastic.out(1,0.3)' });

3 · Tooltip

Contextual help that appears on hover.

Plain · CSS only
Created Apr 12, 2024 Hover for details

Pure CSS — :hover fades and lifts the tooltip. Zero JS.

GSAP
Created Apr 12, 2024 Hover for details

GSAP springs it in with overshoot and eases it out on leave.

Plain — CSS

.tip { opacity: 0;
  transform: translateY(6px);
  transition: opacity .18s, transform .18s; }
.wrap:hover .tip {
  opacity: 1; transform: translateY(0); }

GSAP — JS

wrap.addEventListener('mouseenter', () =>
  gsap.to(tip, { opacity: 1, y: 0,
    ease: 'back.out(3)' }));
wrap.addEventListener('mouseleave', () =>
  gsap.to(tip, { opacity: 0, y: 6 }));

4 · Modal dialog

The backdrop fades in as the dialog scales up to meet you.

Plain · CSS transition

Toggling .open drives a CSS fade + scale.

GSAP

GSAP fades the backdrop and springs the dialog with a settle.

Plain — CSS

.backdrop { opacity: 0; transition: opacity .25s; }
.backdrop.open { opacity: 1; }
.backdrop .dialog { transform: scale(.92); }
.backdrop.open .dialog { transform: scale(1); }

GSAP — timeline

gsap.timeline()
  .to(backdrop, { opacity: 1, duration: .25 })
  .from(dialog, { scale: .9, y: 20, opacity: 0,
        ease: 'back.out(1.7)' }, '<');
