Toggles & selection

Controls that flip state — and reward the flip with a little motion. Click each one. Plain CSS/JS on the left, GSAP on the right.

Plain CSS / vanilla JS GSAP

1 · On / off switch

The knob slides across as the track fills with colour.

Plain · CSS only

A hidden checkbox + :checked sibling selector. Zero JS.

GSAP

GSAP eases the knob with overshoot — easy to add a bounce or squash.

Plain — CSS

.track::after { /* the knob */
  transition: transform .28s cubic-bezier(.2,.8,.3,1.4);
}
input:checked + .track { background: var(--accent); }
input:checked + .track::after {
  transform: translateX(24px);
}

GSAP — JS

sw.addEventListener('click', () => {
  on = !on;
  gsap.to(knob, { x: on ? 24 : 0,
    duration: .35, ease: 'back.out(2)' });
  gsap.to(sw, { backgroundColor: on ? '#6c8cff' : '#1c2130' });
});

2 · Animated checkbox

The checkmark draws itself in stroke by stroke.

Plain · CSS only

An SVG path with stroke-dashoffset animated on :checked.

GSAP

GSAP draws the stroke and pops the box — sequenced on a timeline.

Plain — CSS

.box path {
  stroke-dasharray: 24;
  stroke-dashoffset: 24;   /* hidden */
}
input:checked + .box path {
  transition: stroke-dashoffset .3s ease;
  stroke-dashoffset: 0;     /* drawn */
}

GSAP — timeline

gsap.timeline()
  .to(box, { scale: .85, duration: .1 })
  .to(box, { scale: 1, ease: 'back.out(3)' })
  .to(path, { strokeDashoffset: 0,
        duration: .3 }, '<');

3 · Star rating

Hover to preview, click to set. The everyday review control.

Plain
★★★★★
Rate it

JS lights stars on hover/click; CSS transitions the colour.

GSAP
★★★★★
Rate it

On click, GSAP pops the selected stars with a staggered bounce.

Plain — JS

stars.forEach((s, i) => {
  s.onmouseenter = () => light(i);
  s.onclick      = () => { rating = i; light(i); };
});
// light(): toggle .lit on stars 0..i

GSAP — JS

s.onclick = () => {
  rating = i; light(i);
  gsap.fromTo(stars.slice(0, i+1),
    { scale: .5 },
    { scale: 1, stagger: .06, ease: 'back.out(4)' });
};

4 · Segmented control

A sliding indicator glides to the chosen option.

Plain · CSS transition

JS moves the indicator; a CSS transition smooths the slide.

GSAP

GSAP slides the indicator with an elastic settle — it overshoots, then lands.

Plain — JS + CSS

ind.style.width = btn.offsetWidth + 'px';
ind.style.transform = `translateX(${btn.offsetLeft - 4}px)`;

/* CSS */
.seg-ind { transition: transform .3s, width .3s; }

GSAP — JS

gsap.to(ind, {
  width: btn.offsetWidth,
  x: btn.offsetLeft - 4,
  duration: .5, ease: 'elastic.out(1, 0.6)'
});
