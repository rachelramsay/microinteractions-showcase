Loading & progress

Feedback for the in-between moments — the difference between “is this broken?” and “it’s working.” Trigger each demo with its button. Plain CSS/JS on the left, GSAP on the right.

Plain CSS / vanilla JS GSAP

1 · Skeleton screen

Shimmering placeholders stand in until the real content loads.

Plain · CSS shimmer

A CSS gradient animates its background-position for the shimmer.

GSAP

GSAP cross-fades skeleton → content and staggers the lines in.

Plain — CSS

.shimmer {
  background: linear-gradient(90deg,
    #1c2130, #2c3346, #1c2130);
  background-size: 200% 100%;
  animation: shimmer 1.3s linear infinite;
}
@keyframes shimmer {
  to { background-position: -200% 0; } }

GSAP — JS

gsap.timeline()
  .to(skeleton, { opacity: 0, duration: .3 })
  .set(skeleton, { display: 'none' })
  .from(contentLines, { opacity: 0, y: 8,
        stagger: .08, ease: 'power2.out' });

2 · Progress bar

A determinate bar that fills toward 100%.

Plain · CSS transition
Uploading…0%

JS bumps the width in uneven steps; CSS smooths each jump.

GSAP
Uploading…0%

One eased tween drives both the width and the % readout via onUpdate.

Plain — JS

let p = 0;
const id = setInterval(() => {
  p = Math.min(100, p + Math.random()*18);
  fill.style.width = p + '%';
  pct.textContent = Math.round(p) + '%';
  if (p >= 100) clearInterval(id);
}, 250);

GSAP — JS

const o = { p: 0 };
gsap.to(o, { p: 100, duration: 2.4, ease: 'power1.inOut',
  onUpdate() {
    fill.style.width = o.p + '%';
    pct.textContent = Math.round(o.p) + '%';
  } });

3 · Indeterminate spinner

For when you can’t measure progress — just show it’s working.

Plain · CSS only

A bordered circle rotated at a constant speed by CSS. Zero JS.

GSAP

GSAP spins the SVG and breathes the arc length — that lively “Material” feel.

Plain — CSS

.spinner {
  border: 4px solid #1c2130;
  border-top-color: var(--accent);
  border-radius: 50%;
  animation: spin .8s linear infinite;
}
@keyframes spin { to { transform: rotate(360deg); } }

GSAP — JS

gsap.to(svg, { rotation: 360, repeat: -1,
  duration: 1, ease: 'none', transformOrigin: 'center' });

gsap.fromTo(circle,
  { strokeDasharray: '1 200', strokeDashoffset: 0 },
  { strokeDasharray: '90 200', strokeDashoffset: -35,
    repeat: -1, yoyo: true, duration: .8 });

4 · Animated number counter

A stat that rolls up from zero — great for dashboards and landing pages.

Plain · requestAnimationFrame
0

A hand-written rAF loop with a manual easing curve.

GSAP
0

GSAP tweens a value with built-in easing and snap — one call.

Plain — JS

const start = performance.now();
function tick(now) {
  let t = Math.min(1, (now - start) / 1800);
  t = 1 - Math.pow(1 - t, 3);   // easeOutCubic
  el.textContent = Math.round(t * 2847).toLocaleString();
  if (t < 1) requestAnimationFrame(tick);
}
requestAnimationFrame(tick);

GSAP — JS

const o = { v: 0 };
gsap.to(o, { v: 2847, duration: 1.8, ease: 'power3.out',
  snap: { v: 1 },
  onUpdate: () => el.textContent = o.v.toLocaleString() });
