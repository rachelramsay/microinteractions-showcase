Delight

The optional flourishes — used sparingly, they’re what people remember. This is where a motion library really pulls ahead of plain CSS. Plain on the left, GSAP on the right.

Plain CSS / vanilla JS GSAP

1 · Cursor follower

A dot trails the pointer. Move your mouse inside each box.

Plain · CSS lag
move inside

JS sets x/y; a CSS transition creates the trailing lag.

GSAP
move inside

gsap.quickTo gives a springy, frame-perfect trail with tunable easing.

Plain — CSS + JS

.follower { transition: left .12s, top .12s; }

stage.onmousemove = (e) => {
  dot.style.left = e.offsetX + 'px';
  dot.style.top  = e.offsetY + 'px';
};

GSAP — quickTo

const xTo = gsap.quickTo(dot, 'left', { duration: .4, ease: 'power3' });
const yTo = gsap.quickTo(dot, 'top',  { duration: .4, ease: 'power3' });
stage.onmousemove = (e) => { xTo(e.offsetX); yTo(e.offsetY); };

2 · Success confetti

A celebratory burst for a completed action.

Plain · CSS transition

JS spawns pieces, then a CSS transition flings each to a random spot.

GSAP

GSAP gives each piece arc, gravity, spin and fade — physics CSS can’t express.

Plain — JS

piece.style.transition = 'transform 1s, opacity 1s';
requestAnimationFrame(() => {
  piece.style.transform =
    `translate(${rx}px, ${ry}px) rotate(${r}deg)`;
  piece.style.opacity = 0;
});

GSAP — JS

gsap.to(piece, {
  x: rx, y: ry, rotation: r * 3,
  opacity: 0, duration: 1 + Math.random(),
  ease: 'power2.out',   // arc + gravity feel
  onComplete: () => piece.remove()
});

3 · Magnetic button

The button leans toward your cursor as you approach.

Plain · CSS only

CSS can grow on hover — but it can’t be pulled toward the pointer.

GSAP

GSAP translates the button toward the cursor, then snaps back elastically.

Plain — CSS

.mag-btn:hover {
  transform: scale(1.06);
  box-shadow: 0 12px 26px -10px rgba(108,140,255,.8);
}
// no pointer-following in CSS

GSAP — JS

btn.onmousemove = (e) => {
  const r = btn.getBoundingClientRect();
  gsap.to(btn, { x: (e.clientX - r.left - r.width/2) * .4,
    y: (e.clientY - r.top - r.height/2) * .4, duration: .4 });
};
btn.onmouseleave = () => gsap.to(btn,
  { x: 0, y: 0, ease: 'elastic.out(1, 0.3)' });

4 · Animated empty state

A gentle idle motion keeps an otherwise dead screen feeling alive.

Plain · CSS loop
📭
No messages yet
New mail will appear here.

A single CSS @keyframes bobs the icon up and down forever.

GSAP
📭
No messages yet
New mail will appear here.

GSAP floats the icon and staggers the text in on load — one timeline.

Plain — CSS

.glyph-box { animation: float 2.6s ease-in-out infinite; }
@keyframes float {
  0%, 100% { transform: translateY(0); }
  50%      { transform: translateY(-9px); }
}

GSAP — JS

gsap.from([icon, title, sub],
  { opacity: 0, y: 12, stagger: .12 });   // entrance

gsap.to(icon, { y: -9, repeat: -1, yoyo: true,
  duration: 1.3, ease: 'sine.inOut' });   // idle float
