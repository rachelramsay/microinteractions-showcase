How an interface responds to moving through it — menus, headers, and scroll position. Interact with each. Plain CSS/JS on the left, GSAP on the right.

1 · Hamburger → X morph

The menu icon folds into a close icon. Click it.

Plain · CSS only

An .open class rotates the lines via CSS transitions.

GSAP

A reversible GSAP timeline morphs the lines — play forward, reverse to undo.

Plain — CSS

.open .ln1 { transform: translateY(0) rotate(45deg); }
.open .ln2 { opacity: 0; }
.open .ln3 { transform: translateY(0) rotate(-45deg); }

GSAP — timeline

const tl = gsap.timeline({ paused: true })
  .to(ln1, { y: 0, rotate: 45 }, 0)
  .to(ln2, { opacity: 0 }, 0)
  .to(ln3, { y: 0, rotate: -45 }, 0);
btn.onclick = () => open ? tl.reverse() : tl.play();

Plain — JS

});

GSAP — JS

box.addEventListener('scroll', () => {
  const small = box.scrollTop > 20;
  gsap.to(header, { paddingTop: small ? 9 : 18,
    fontSize: small ? 14 : 18, duration: .3 });
});

gsap.quickTo smooths the bar — it glides to catch up with the scroll.

Plain — JS

box.onscroll = () => {
  const max = box.scrollHeight - box.clientHeight;
  fill.style.width = (box.scrollTop / max * 100) + '%';
};

GSAP — quickTo

const set = gsap.quickTo(fill, 'width',
  { duration: .3, ease: 'power3' });
box.onscroll = () => {
  const max = box.scrollHeight - box.clientHeight;
  set((box.scrollTop / max * 100) + '%');
};

4 · Accordion

A panel expands smoothly to reveal its content.

Plain · CSS grid-rows
A small, single-purpose moment of feedback — a toggle, a like, a loading state.
They make an interface feel responsive, alive, and trustworthy.

A CSS grid-template-rows: 0fr → 1fr transition — no JS height math.

GSAP
A small, single-purpose moment of feedback — a toggle, a like, a loading state.
They make an interface feel responsive, alive, and trustworthy.

GSAP animates height: 0 → auto with easing — and can stagger content in.

Plain — CSS

.acc-panel {
  display: grid; grid-template-rows: 0fr;
  transition: grid-template-rows .3s ease;
}
.open .acc-panel { grid-template-rows: 1fr; }
.acc-panel > div { overflow: hidden; }

GSAP — JS

open
  ? gsap.to(panel, { height: 'auto', duration: .35,
        ease: 'power2.out' })
  : gsap.to(panel, { height: 0, duration: .3 });
