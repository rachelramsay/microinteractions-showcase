Common web microinteractions

Microinteractions are the small, single-purpose moments of feedback that make an interface feel responsive — a button that lifts, an input that confirms it's valid, a toast that slides in. Each demo here is built two ways: hand-rolled with plain CSS & vanilla JS, and again with the GSAP animation library, so you can compare the effort and the result. Toggle “View code” on any demo to see the source.
